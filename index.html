Research Work

Computer Vision · Traffic Analysis

Hybrid CLIP-YOLO Framework for Traffic Density Classification in Urban Environments

Developed a dual-stream traffic density classification framework combining CLIP-based global scene embeddings with YOLOv8 vehicle-type counts, fused using XGBoost for robust density prediction in non-lane-based traffic. Demonstrates a cost-effective solution deployable on existing CCTV infrastructure with real-time inference on both GPU and CPU.

94.25% Accuracy
CLIP + YOLOv8
XGBoost Fusion Model
NLP · Mental Health AI

AI-Driven Speech-Based Depression Detection

Developed a Multi-Scale Cross-Modal Transformer (MSCMT) to detect depression by fusing 88 acoustic features, 768-D semantic embeddings, and behavioral markers. Optimised a multi-stage pipeline featuring speaker diarization, Mutual Information-based feature selection, and a stacked XGBoost ensemble. Validated through feature importance analysis — deep latent embeddings provided 73.9% of the predictive signal.

84.21% Accuracy
73.9% Embedding Signal
MSCMT Architecture
